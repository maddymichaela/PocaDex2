# PocaDex

PocaDex is a web binder for K-pop photocard collectors. It helps collectors organize owned cards, wishlists, duplicates, and incoming cards with searchable details, images, and profile-sharing features.

## Features

- Personal collection tracking for owned, wishlist, duplicate, and on-the-way cards
- Binder views grouped by artist, member, era, category, or year
- Search, filters, sorting, and collection progress tracking
- Card image upload with crop, rotation, and zoom tools
- Scan/import flow for splitting grid template images into individual card crops
- Dashboard with collection counts, progress, duplicates, groups, and recent additions
- Public collector profiles, user discovery, and follow/following lists
- JSON export and import for collection backups
- Account settings with avatar upload and privacy controls

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Supabase Auth, Database, and Storage
- react-easy-crop
- lucide-react
- Motion
